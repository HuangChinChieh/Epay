Encryption Method

The encryption method is SHA256.

After encryption, it needs to be converted to uppercase.

php:

        
                hash('sha256', $data, false);

java:

        
                MessageDigest digest = MessageDigest.getInstance("SHA-256");
                digest.reset();
                digest.update(input.getBytes("utf8"));
                toReturn = String.format("%064x", new BigInteger(1, digest.digest()));

encrypted string:

5566520

Encrypted result:

69481E4583F7A8FBC1200BC629603CB9A0D5B16822DE87B0ACA965C89CD80F00

Online testing website URL:https://emn178.github.io/online-tools/sha256.html

Payment Create

URL: {APIURL}/RequirePayingUrl

Method: Post

Content-Type: application/json

Request :

PS : The signature is encrypted using SHA256.

Parameter Type Required Description
ManageCode String YES Merchant Code
Currency String YES    Currency (see comparison table for details)
Service String YES    Service type (see comparison table for details)
CustomerIP String YES Member IP
OrderID String YES Merchant Order Number
OrderDate String YES Order Date
The required date string format is "yyyy-MM-dd HH:mm:ss", with an example being "ex:2018-01-01 11:22:33".
OrderAmount Number YES Merchant Order Amount
Only supports integers
RevolveURL String YES Callback URL
UserName String NO Depositor Name
If it is a real-name system channel, it needs to be filled in
State String NO If the merchant passes this parameter when making a payment request, the parameter will be returned to the merchant in the same format when the payment is successful (allowing the value to be an empty string, for example: State="")
Sign String YES Sign:translates to "Needs to be encrypted with SHA256 and converted to uppercase
If OrderAmount has decimals, please remove the trailing zeros. For example, for 100.00, please include 100 (ex: 100.00->100) in the signature.
The signature format should follow the example. Only include these parameters in the signature string:
ManageCode=TestCode&Currency=CNY&Service=AA001&OrderID=OID001&OrderAmount=100&OrderDate=2020-01-01 01:01:01&CompanyKey=APIKey 
        

                {
                    ManageCode:'ManageCode',
                    Currency:'CNY',
                    Service:'AA001',
                    CustomerIP:'111.111.11.111',
                    OrderID:'OID001',
                    OrderDate:'2020-01-01 01:01:01',
                    OrderAmount:100.00,
                    RevolveURL:'http://ReturnURL',
                    Sign:'ManageCode=ManageCode&Currency=CNY&Service=AA001&OrderID=OID001&OrderAmount=100&OrderDate=2020-01-01 01:01:01&CompanyKey=CompanyKey'
                }
